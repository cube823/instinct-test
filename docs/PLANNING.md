# 본능 테스트 - 생존/번식 성격 유형 검사

## 1. 컨셉

인간의 두 가지 근본 욕구인 **생존 본능**과 **번식 본능**을 축으로, 어느 쪽 비중이 높은지 + 각각의 강도를 측정하는 성격 유형 검사.

- **생존 본능**: 안정 추구, 리스크 회피, 자기보존, 경쟁심, 독립성, 자원 확보
- **번식 본능**: 매력 과시, 관계 형성, 사회적 인정 욕구, 양육 본능, 짝짓기 디스플레이

### 레퍼런스

- 테토/에겐 검사법 (테스토스테론/에스트로겐 기반 성격 유형)
- 2025년 한국에서 크게 유행, 밈화된 성격 테스트

### 바이럴 전략

- "번식"이라는 직설적 단어 자체가 밈 포인트
- 테스트 소개에서는 "관계 본능", "사회적 끌림" 등으로 부드럽게 설명
- 결과는 대놓고 **미친번식남** 같은 파격적 네이밍 → 캡처 & 공유 유도

---

## 2. 결과 유형 시스템

### 네이밍 구조: [강도] + [유형] + [성별]

| 강도 | 조건 | 예시 |
|------|------|------|
| **미친~** | 우세 축 45+ & 차이 15+ | 미친생존남, 미친번식녀 |
| **찐~** | 우세 축 35+ & 차이 10+ | 찐생존녀, 찐번식남 |
| **반반~** | 차이 5~9 | 반반생존남 (약간 우세 쪽 표기) |
| **균형형** | 차이 0~4 | 균형형 |

### 점수 산출

- 생존축 10문항, 번식축 10문항 (총 20문항)
- 1-5 스케일: 그대로 1~5점
- Y/N: Yes = 5점 / No = 1점
- 각 축 점수 범위: 10~50

## 3. 문항 설계

### 생존축 (10문항)

| # | 문항 | 형식 |
|---|------|------|
| S1 | 통장에 비상금이 없으면 불안해서 잠이 안 온다 | 1-5 |
| S2 | 길을 걸을 때 무의식적으로 도망칠 경로를 파악하고 있다 | 1-5 |
| S3 | 좀비 아포칼립스가 오면 나는 끝까지 살아남을 자신이 있다 | Y/N |
| S4 | 먹고 싶은 것보다 몸에 좋은 것을 먼저 고른다 | 1-5 |
| S5 | 경쟁에서 지면 그날 밤 잠이 안 온다 | 1-5 |
| S6 | 위험해 보이는 골목길은 돌아가더라도 피한다 | 1-5 |
| S7 | 여행 전에 보험, 비상약, 대사관 번호를 미리 다 챙긴다 | Y/N |
| S8 | 혼자서도 충분히 잘 살 수 있다는 확신이 있다 | 1-5 |
| S9 | 10년 뒤의 안정이 오늘의 즐거움보다 중요하다 | 1-5 |
| S10 | 세상이 망해도 나만은 살아남을 계획이 있다 | 1-5 |

### 번식축 (10문항)

| # | 문항 | 형식 |
|---|------|------|
| B1 | 외출 전 거울을 안 보면 하루가 불안하다 | 1-5 |
| B2 | 좋아하는 사람 앞에서 나도 모르게 다른 사람이 된다 | 1-5 |
| B3 | SNS에 올린 사진 반응이 별로면 슬쩍 삭제한 적이 있다 | Y/N |
| B4 | 매력적인 사람이 옆에 있으면 나도 모르게 자세를 고쳐 앉는다 | 1-5 |
| B5 | 향수, 옷, 헤어 등 외모 관리에 월 10만 원 이상 쓴다 | Y/N |
| B6 | "매력 있다"는 말 한마디면 한 달은 기분 좋게 산다 | 1-5 |
| B7 | 연애 상대의 유전적 조건(키, 외모, 건강)이 솔직히 중요하다 | 1-5 |
| B8 | 사람들 사이에서 주목받고 싶은 욕구가 있다 | 1-5 |
| B9 | 좋아하는 사람의 이성 친구가 은근히 신경 쓰인다 | 1-5 |
| B10 | 나는 언젠가 꽤 괜찮은 부모가 될 자신이 있다 | 1-5 |

### 출제 순서 (실제 테스트에서)

축이 드러나지 않도록 교차 배치:

```
1.  S3  (좀비 - 가볍게 시작)
2.  B1  (거울)
3.  S1  (통장)
4.  B2  (좋아하는 사람 앞에서)
5.  S6  (골목길)
6.  B3  (SNS 삭제)
7.  S2  (도망 경로)
8.  B4  (자세 고쳐앉기)
9.  S4  (몸에 좋은 것)
10. B6  (매력 있다는 말)
11. S5  (경쟁에서 지면)
12. B5  (월 10만원)
13. S8  (혼자 잘 살 수 있다)
14. B9  (이성 친구 신경)
15. S9  (10년 뒤 안정)
16. B7  (유전적 조건) ← 후반부에 도발적 문항
17. S7  (여행 준비)
18. B8  (주목받고 싶다)
19. S10 (세상 망해도)
20. B10 (좋은 부모) ← 진지하게 마무리
```

## 4. 결과 유형 상세

### 미친생존 (남/녀)

**한줄 소개**: "핵전쟁이 나도 마지막까지 살아남을 사람"

**특징**
- 통장 잔고 = 심장 박동수. 비상금 없으면 산소가 끊긴 느낌
- 건강검진 예약은 새해 첫 번째 할 일
- "위험해 보이는데?" 이 말이 입에 붙어 있음
- 혼자서 무인도에 떨어져도 3일 안에 집을 짓고 있을 사람
- 계획 없는 여행? 그건 여행이 아니라 조난

**연애 스타일**
- "먼저 각자 경제적으로 독립하고, 그다음에 만나자"
- 감정보다 안정, 설렘보다 신뢰
- 연애는 하고 싶지만 내 생존 시스템을 흔들 정도는 NO

**명대사**: "사랑으로 밥을 먹을 수 있어?"

---

### 찐생존 (남/녀)

**한줄 소개**: "조용하지만 확실한 서바이버"

**특징**
- 적당히 아끼고, 적당히 조심하고, 근데 그게 다 계산된 거임
- 보험 3개 이상 가입, 근데 남한테 자랑 안 함
- "일단 안전한 선택지"가 기본 세팅
- 모험? 가끔은 하지만, 안전장치 확인 후에
- 사람은 좋아하는데 결국 혼자가 편함

**연애 스타일**
- 안정적인 관계를 원함. 롤러코스터 연애는 사절
- 상대에게 "너 보험 있어?" 라고 물어본 적 있음 (비유적으로)
- 느리지만 한번 믿으면 끝까지 감

**명대사**: "좋아는 하는데... 일단 내 할 일 먼저 하고"

---

### 미친번식 (남/녀)

**한줄 소개**: "존재 자체가 페로몬, 걸어다니는 끌림 본능"

**특징**
- 외출 준비 시간 최소 1시간. 향수는 무기
- SNS 프로필 사진 교체 주기 = 계절 바뀔 때마다
- 매력적인 사람 반경 3m 진입 시 자동으로 매력 모드 ON
- "좋아요" 알림이 도파민 공급원
- 사람들 사이에서 빛나고 있어야 살아있는 느낌

**연애 스타일**
- 항상 누군가를 좋아하고 있거나, 좋아할 사람을 찾는 중
- 밀당은 본능, 눈 맞춤은 기술
- 연애 감정이 인생 에너지의 80%를 차지

**명대사**: "사는 게 뭐야, 사랑해야 사는 거지"

---

### 찐번식 (남/녀)

**한줄 소개**: "자연스럽게 사람을 끌어모으는 인간 자석"

**특징**
- 의식적으로 꾸미진 않는데 신경은 씀
- 사람 만나는 걸 좋아하고, 관계를 잘 유지함
- "너 매력 있다" 한마디에 일주일 충전 완료
- 상대의 외모도 보지만 인정하기 싫어함
- 좋아하는 사람 생기면 생활 패턴이 바뀜

**연애 스타일**
- 연애 중요하게 생각하지만 인생 전부를 걸진 않음
- 자연스러운 만남 선호, 억지 소개팅은 어색
- 한번 빠지면 깊이 빠지는데, 티를 안 내려고 노력

**명대사**: "관심 없는 척하는데... 아 근데 그 사람 인스타 뭐지?"

---

### 반반형 (남/녀)

**한줄 소개**: "상황에 따라 생존 모드 ↔ 번식 모드 전환하는 하이브리드"

**특징**
- 평소엔 현실적인데, 좋아하는 사람 앞에서 갑자기 로맨티스트
- 돈 아끼다가도 데이트 때는 지갑이 열림
- 혼자도 좋고, 같이도 좋음. 근데 선택하라면 고민 5시간
- 양쪽 유형 친구한테 다 공감됨
- "나 이거 맞는 거 같기도 하고 아닌 거 같기도 하고"

**연애 스타일**
- 연애와 개인 생활의 밸런스를 중시
- 때로는 생존형처럼 이성적, 때로는 번식형처럼 감성적
- 상대 유형에 따라 자기도 모르게 맞춰가는 카멜레온

**명대사**: "둘 다 중요하지 않아? 왜 하나만 골라"

---

### 균형형 (남/녀)

**한줄 소개**: "생존과 번식의 완벽 밸런스, 진화가 설계한 최적의 인간"

**특징**
- 생존 본능과 번식 본능이 거의 정확히 50:50
- 현실적이면서 로맨틱, 조심스러우면서 매력적
- 어떤 상황에서든 적응 가능한 올라운더
- "너는 좀 특이하다"는 말을 자주 들음
- 양쪽 극단 유형들이 부러워하는 타입

**연애 스타일**
- 이상적인 파트너상이 명확하면서도 유연함
- 연애도 즐기고, 개인 성장도 놓치지 않음
- 의외로 이 유형이 가장 오래가는 관계를 만듦

**명대사**: "나? 그냥 적당히 다 하는 편인데?"

## 6. 기술 스택

- **프레임워크**: Next.js (App Router, TypeScript)
- **스타일링**: Tailwind CSS
- **결과 카드 이미지**: html2canvas 또는 @vercel/og (OG Image Generation)
- **배포**: Vercel

